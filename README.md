# Femboy-Detector
Random thing i made in my free time. Read the ReadMe for more info.

So i started this project when i was bored and remembered I have a kit for an Arduino.

For this to work perfectly you'll need:
An Arduino UNO (i belive any works)
The RFID-RC522 module
An IIC 1602 LCD Screen
An RFID tag
Lots of jumper cables
Lots of free will

Firstly connect the 5V to the breadboard's "+" section and the GND to the "-" section.
Then connect the screen,
GND to the "-" section
VCC to the "+" section
SDA to the Arduino's A4
SCL to the Arduino's A5

Then connect the RFID module (gl bro)
RST to Pin 9
SDA to Pin 10
MOSI to Pin 11
MISO to Pin 12
SCK to Pin 13

Connect your Arduino and run the code in the file 👍
